Thank you for downloading the 99 Mods on the Wall: Fabric Edition server!

1. Make sure you have Java 17 installed

2. To download and install fabric, read https://fabricmc.net/use/?page=server

3. To start the server:
 - Using Mac or Linux run java -Xms6G -Xmx6G -jar fabric-server.jar nogui
 - Windows you can double click the "fabric-server.jar" and your server will power up

4. You will then need to open the EULA and change from eula=false to eula=true
